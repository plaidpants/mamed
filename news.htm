June 9, 2002

Sorry for the lack of updates but this is just a hobby after all. Speaking of hobbies I spent some time in San Francisco recently at the recent battlebots competition.

First, another programmable camera has been discovered and someone is working on hacking it to maybe one day run MAME. You can read the post here and go to his web site here. If anyone has one of these cameras and want to help out I'm sure he would appreciate it.

Also since the demise of the flashpoint website you can still find some digita information at the Digita World site here.

There have been some new versions of MAME available for the dreamcast floating about recently. Sesei has done some amazing working figuring out the mame core and the MAMED and mamedc dreamcast ports to make his own port of MAME. You can download his CP1MAME37b16 version here. He has some source and other info available but that part of his web site appears to be missing right now and he has deleted his posts from the message boards. Hopefully we will see it return soon. Everyone please be nice to each other and work together a little.

Others have used this Sesei code to create several other versions of MAME. You can check out warmtoe site here for more info on his version. WarmToe was recently able to recode the mame sound code driver to directly access the dreamcast sound HW and should allow for much faster sound emulation. He has been currently working on a Williams version called MAIM. There is also some information on some other ports here at the Unofficial MAMED American Website.

Also the original MAMED compatibility site seems to has disappeared, Quicksilver was kind enough to send me the tables & scripts from the site so they are not lost. I will try and host them here when I get a chance. In the meantime you can check out a new different compatibility site here, thanks to droogieboy for pointing it out to me and emuxer for creating it.

Kudos for everyone working on making some new versions of the MAME that runs on the dreamcast.

Finally I added to the download page a 37b6.5 version of MAMED for download. This is an older mame core which seemed to run some games which the 37b15 did not want to run. However this version has the updated UI core from the 37b15.5 version. I built and tested this a long time ago but never got around to uploading it, please let me know if it works, if it doesn't I just remove it. Also a reuploaded the original b37.15 cdi image with the rar recovery record so hopefully people will be able to repair a bad download, maybe I should split it and have par files.
