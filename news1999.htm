I am also featured in the FlashPoint Digita Developer Spotlight along with my friend Scott Angel.

Nov 21,1999

New version 36b91b available that supports high resolution games. There is also a game options menu for quick selection of pad orientation and display rotation. Slightly different set of games supported, as usual. Used a larger font for better reading but it gets small in high resolutiuon mode. This version is based on 36b91b version of MAME source code. The new MAME licence agreement requires that MAMED have the same version number as the MAME code it was based on, thus the new version number.

Sound is not implemented in this version but here is an update for you. Sound code is coming along well, it finally sounds right, thanks to XMAME, the unix version of MAME. It's still is choppy due to some low level driver digita issues which we are going to have to rewrite to correct. Hopefully soon.

I am working on the memory issues and I finally have some working test code by reconstructing the system symbol table. Now that I've got the high resolution and rotation stuff working I will try and cut up MAME in nice easy chunks that I can load and link together nicely. Some of the game drivers are cross linked so it's not so easy to find the best way to break them all up.

Sorry this last release took so long, had a little partition corruption problem on the PC after trying to install Linux. The MAMED source code was safe, but my MAME drive was toast though. Oh well.

Enjoy.
