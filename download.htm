MAME for Dreamcast (MAMED)

Each MAMED for dreamcast pack supports different roms. I would suggest downloading the cdi file and burning it with DJ and then using a separate roms disk. To burn a self boot disk with roms use the plain files with the special version of KOS for MAMED using the self boot kit. You can get lot's of help burning selfboot dreamcast games here. You can read Hyper Gamer's MAMED FAQ here. Finally the unoffical MAMED compatability list is here. Also don't forget about the message boards here and the ReadMe here.
