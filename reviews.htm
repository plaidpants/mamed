Digita OS Digital Cameras

These are currently the only shipping cameras available that have the Digita OS open operating system that applications like MAMED can be written for. No other camera support this, if it's not on this list you are out of luck. I really mean it, if it's not here you cannot run MAMED or any other Digita OS application.

If you'd like to see MAMED on your camera and it's not on this list let that camera manufacturer know about it.

Digita Camera Perfomance Measurements

Measured using version 0.36rc1 with pacman. Other games and versions will have different performance characteristics. Kodak DC265 performance was almost exactly the same as the Kodak DC290. I wasn't able to get my hands on a Kodak DC260 but I suspect it is somewhere between the DC265 and DC220.

Kodak DC220

66Mhz PowerPC MPC823, 6MB RAM, 1152 x 864, 2x optical zoom, 11khz sound, joypad controller

This is one of the first Digita OS cameras, it doesn't have as many features as the DC260 but it can run applications if there are not too big. The RAM is also a bit slower than the other Kodak's so games run just a tad slower on this Kodak.

Kodak DC260

66Mhz PowerPC MPC823, 8MB RAM, 1536 x 1024, 3x optical zoom, 11khz sound, joypad controller

This is also one of the first Digita OS camera, it has more features and more RAM, especially useful to run application.

Kodak DC265

66Mhz PowerPC MPC823, 16MB RAM, 1536 x 1024, 3x optical zoom, 11khz sound, joypad controller

This was the second generation of the Kodak DC series, it has double the RAM of the previous one.

Kodak DC290

66Mhz PowerPC MPC823, 16MB RAM, 1792 x 1200, 3x optical zoom, 11khz sound, joypad controller

This is the third generation of the Kodak DC series and the second generation of Digita OS, it support a much larger CCD. Sound is now working as of firmware version 1.0.4.

Minolta Dimage 1500 EX

54Mhz DCAM, 16MB RAM, 1344 x 1008, 3x optical zoom, 4 button directional

This was the second platform that Digita OS was ported to. It runs on a different processor than the Kodaks, its based on MIPS and has HW image processing. It has slightly more pixels on the screen in normal mode but they are not square. This camera has the unquie ability to change lens. There are two lens available, a wide angle and a telephoto. The camera can be purchased with one or the other.

Minolta 1500 3D

54Mhz DCAM, 16MB RAM, 1344 x 1008, 3x optical zoom, 4 button directional

This was basically the same camera as the above 1500EX but has a special strobe unit and different firmware inside the camera. It can take 3D pictures by taking two photos from the same lens in short succession. The first photo is normal but the second uses a special filter over the strobe with vertical colored bars. Since the strobe unit is offset from the lens it casts colored bar shadows on the object photographed. These two photos can be processed to produce a true 3D model of the object photographed. The 3D model created is accurate to 1mm for objects close to the lens but less accurate further away. Its kind of cool but pretty expensive.

PENTAX EI-200

90MHz Coldfire, 16MB RAM, 2.1 mega-pixel, 3x optical zoom, joypad directional, sound

This is the latest addition to the Digita OS family. This speed of MAME is similar to the Kodak DC290 but I have not run any actual tests yet. When I get sound working this should be a great game camera. The Coldfire processor is similar to the 68000 processor.

PENTAX EI-2000

90Mhz Coldfire, 16MB RAM, 2.24 mega-pixel, 3x optical zoom, joypad directional, sound

This SLR camera is the latest addition to the Digita OS family currently shipping in Japan. The HP 912 is available in the US and is basically the same camera. This camera has more buttons than any digital camera I have ever seen. It also has a flip up LCD in the back. The camera is kindof bulky for playing games but they run fast.

HP Photosmart C500

54Mhz DCAM, 16MB RAM, 1600 X 1200, 3x optical zoom, joypad directional

This is a recent addition to the Digita OS family. It has a much higher resolution CCD than the previous DCAM based cameras. It takes great photos but is a little slow for playing games.

HP Photosmart 618

90Mhz Coldfire, 16MB RAM, 2.1 mega-pixel, 3x optical zoom, joypad directional, sound

This is the latest addition to the Digita OS family and is basically the same as the PENTAX EI-200.

HP Photosmart 912

90Mhz Coldfire, 16MB RAM, 2.24 mega-pixel, 3x optical zoom, joypad directional, sound

This SLR camera is the latest addition to the Digita OS family and is basically the same as the PENTAX EI-2000.
