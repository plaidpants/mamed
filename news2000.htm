We're coming up on one year old so I better do a new release. New version of MAMED & MESSD is now available on the downloads page for the Kodaks, other versions to come. Lot's of nifty new functionality and of course based on a more recent version of MAME & MESS source code.

SNAP menu support
The main game menu now displays the snap png file for the selected game behind the menu. Just use the shutter button while playing a game to save a png file in the snap directory and then it will appear in the game menu.
MAME.CFG/MESS.CFG support
I now read a config file that can be used to set various MAME/MESS parameters. Edit the MAME.CFG/MESS.CFG file for more information on specific settings available.
CFG directory support
You can configure the CFG directory in the MAME.CFG/MESS.CFG file and then input key setting will be saved for each game individually. Default keys will be saved as well. Also, you will only have to enter OK the first time you play a game.
History and info support
Just configure the HISTORY.DAT/SYSINFO.DAT file in your MAME.CFG/MESS.CFG and now you have a portable video game history museum in your hand with all sorts of nifty historical facts about the video games you are playing. You can also configure the MAMEINFO.DAT file. You can get the current MAMEINFO.DATA and HISTORY.DAT/SYSINFO.DAT file from www.mame.net/mess.emuverse.com.
Hiscore support
Just configure the hi directory and the HISCORE.DAT file in your MAME.CFG and supported games will now remember the hiscore the next time you play. You can get the current HISCORE.DAT file from www.mame.net.
Cheat support
Just turn on and configure cheat in the MAME.CFG file and put a current CHEAT.DAT file on the card. Many cheat options are available for many games. You can also adjust overclock/underclock of the various CPUs in a game and get a little extra performance. Of course hiscore saving is disabled while cheating. You can get the current CHEAT.DAT file from www.mame.net.
Language file support
I now support the .lng files, just set the language file you want to use in the MAME.CFG/MESS.CFG file and all text in MAME/MESS will be translated according to the file. Does anyone have a current source of these files?
Two new screen modes
Two new screen modes (Expand & Stretch) are available and are useful for the Apple // emulation and the Atari 2600 emulation. You will find them in the resolution setting.
Keyboard support
Keyboards are supported through the Kodak serial port. Probably could hook up a joystick with this or even do network stuff. Simply start HyperTerm or whatever and type out the serial port of you PC and into the camera serial port. I convert control characters and capital letters back into shift and control key combinations. Of course you can only type serially, no multiple keys or diagonals here but useful enough to program an Apple //.
I am working on support for keycodes directly sent to the camera from various portable keyboards but it's not quite ready yet.
You can get a serial cable from Kodak for your Digita camera if you ask them nicely.
LED support
From the "why not" department. Now you can see those player one and player two lights when you insert quarters or watch the floppy disk LEDs.
Brightness supported
You can now set the brightness via the on screen menu. Sorry no gamma support yet, just a bit too much floating point math for the camera.
CRC directory supported
MESS image crc files are now read from the crc directory for a little more information about the particular cart or disk you are loading.
Z80 speedup
Changed the z80 cpu emulator to inexact emulation for a little better performance, from the MAMECE team. Everything still seems to work. Don't forget the CPU overclocking available when cheat is enabled from the MAME on-screen menu it can speed some games up a bit by fiddling with the numbers.
Zipped image files work (sort of)
Fixed a bug in the core MESS code so you can now load zipped image files from the file manager menu sometimes. This doesn't always work with some drivers for some reason but at least you can save a few kilobytes by zipping some cart files.

MAMED version 36b14b is now available, it is very different than pervious versions so please read the README. This version DOES NOT work on the DC290 (sorry) however it will now work on the DC220, DC260 and DC265!!! This version includes and supports almost every game except neogeo and the large ROM games, cause there's just not enough RAM for everything whatever I do. Those of you with large compact flash cards will be happy you have them now. I also improved the speed of the sound driver considerably by replacing all floating point with fixed point math.
